# 📝 标准匹配格式 (以"告白气球 - 周杰伦"为例)：
# ✅ 标准格式: "告白气球 - 周杰伦.mp3"
# ✅ 艺术家在前: "周杰伦 - 告白气球.flac"  
# ✅ 下划线分隔: "告白气球_周杰伦.wav"
# ✅ 包含专辑信息: "[十二新作] 周杰伦 - 告白气球.mp3"
# ✅ 包含音质信息: "告白气球 - 周杰伦 [320K].flac"
# ✅ 中英混合: "Jay Chou 周杰伦 - 告白气球.wav"
# ✅ 复杂格式: "2016.07.29 - 周杰伦 - 告白气球 (十二新作).mp3"
# 
# 🔍 匹配原理：只要文件名同时包含"告白气球"和"周杰伦"，就能成功匹配

# ❌ 无法匹配的格式：
# ❌ "气球.mp3" (缺少艺术家信息)

# ❌ "告白-周董.mp3" (艺术家名称不匹配)
